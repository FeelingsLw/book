##### 给数据排序

```python
import numpy as np
a= np.array([[6,4,5],[7,5,3]])
'''
[[6，4，5]
 [7，5，3]]
'''
# 按行排序
b = np.sort(a,axis=1)
'''
[[4，5，6]，
 [3，5，7]]
'''
a=np.array([3,5,6,2,3])
# 将数据的索引排序，返回的是索引
c = np.argsort(a) # [3, 0, 4, 1, 2]
print(a[c]) #[2, 3, 3, 5, 6]
```
